## Technologies
Project is created with:
* React version: 17.0.2
* Redux 
* Eslint version: 8.10.0
* Jest version: 5.16.2
* Node.js
* Firebase version: 9.6.8
## Setup
To run this project, install it locally using npm:
```bash
$ cd ../client
$ npm install
$ npm start
```
